Real Time Multimodal Torpid Detection Using DCCNN

  • The primary objective of the project is to significantly reduce the accident ratio resulting from careless drivers. We aim to leverage cost-effective technology to enhance road safety and minimize the loss of life and property through the use of our web application.
  • It's a low-cost, real-time prototype torpidness detection system with acceptable accuracy. It provide a lightweight alternative to heavier classification models, addressing health concerns while incorporating additional features.
  • The Torpidness Detection System is a proposed mechanism designed to save the lives of drivers who continuously operate cars and buses without adequate sleep. This system is especially critical in developing countries like India, where the number of vehicles on the road increases each year.
  • The system can detect whether the driver's eyes are open or closed, issuing warnings accordingly. The eye detection capability can be enhanced using a combination of different algorithms that leverage edge detection techniques, machine learning concepts, and open-source libraries like OpenCV.
